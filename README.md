## 🪙 Despre Adocoin

Adocoin este un token creat pe rețeaua Polygon, cu o ofertă totală de 21 milioane ADO. Scopul proiectului este să ofere o alternativă cripto românească transparentă, accesibilă și orientată spre viitor.
